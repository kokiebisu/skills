# skills

My agent skills, installable with [`npx skills`](https://github.com/vercel-labs/skills).

## Install

```bash
npx skills add kokiebisu/skills
```

## Skills

- **[onboard](./skills/onboard/SKILL.md)** — Claude-led, interactive guided tour that teaches a developer the domain model of a codebase, to speed up onboarding onto an unfamiliar repo.
- **[setup-onboard](./skills/setup-onboard/SKILL.md)** — One-time, optional setup for `onboard` (doc languages, output directory, default commit behavior).
- **[pr](./skills/pr/SKILL.md)** — Create focused pull requests with changes grouped logically (max 200 lines each). Supports parallel multi-PR workflows via Agent Teams.
